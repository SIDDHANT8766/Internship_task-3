# Registration Form
      A simple and responsive **Registration Form** built using **HTML5** and **CSS3**. This project demonstrates the fundamentals of web forms, semantic HTML, and modern CSS styling.

## 📌 Features
      * Responsive form layout
      * Name, Email, and Password input fields
      * HTML5 form validation using the `required` attribute
      * Clean and modern user interface
      * Hover effect on the submit button
      * Mobile-friendly design

## 🛠️ Technologies Used
      * HTML5
      * CSS3

## 📂 Project Structure
      ```
      Registration-Form/
      │
      ├── index.html

      └── README.md
      ```

## 📖 Project Overview 
      The registration form includes the following fields:
      
      * Name
      * Email Address
      * Password
      * Submit Button
      
      The form is styled using CSS to provide:
      
      * Centered layout
      * Rounded corners
      * Box shadow effect
      * Responsive width
      * Interactive button hover effect
